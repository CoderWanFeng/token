## 本轮改版内容（v1）

### 1. 首页结构调整（src/pages/Home.tsx）
- 移除原「打开就看到 SmartGuide（智能导购）」的主位布局
- 改为：Hero → ProvidersSection（默认展开）→ SmartGuide（折叠，默认收起）→ HowToChoose → Footer
- SmartGuide 收纳到带边框的折叠区里，按钮文案改为「不知道选哪个？打开智能导购帮你选」，提示语改为「回答 3 个问题，自动推荐最划算的 Plan」
- SEO meta 同步调整：title / description / og 全部改为「AI Token 优惠聚合 - 各大云服务商 Coding Plan 一站对比」定位
- 删除原来的「查看全部 16 个 Plan」按钮（已无必要，列表直接展开）

### 2. Hero CTA 调整（src/components/Hero.tsx）
- 主 CTA「立即挑选套餐」锚点从 `#smart-guide` 改为 `#providers`，点选直接滚到优惠列表区
- 次 CTA 文字从「算算我要花多少」改为「帮我选最划算的」，突出「帮用户选择」的语义，与主 CTA 形成「自己挑 / 让 AI 帮我挑」对仗

### 3. Provider 数据类型扩展（src/data/providers.ts）
- `Provider` 接口新增两个可选字段：
  - `discount?: number` —— 折扣角标数值（自动除以 10 格式化）
  - `priceTag?: string` —— 价格标签自定义文案
- 三个 provider 写入首批数据：
  - 阿里云百炼：`discount: 75` → 显示「7.5折」
  - 火山方舟 Token Plan：`priceTag: '低至9.4元'`
  - MiniMax：`discount: 90` → 显示「9折」

### 4. ProviderCard 卡片角标（src/components/ProviderCard.tsx）
- 右上角绝对定位（top-4 right-4），flex 列向堆叠
- 折扣角标：红橙渐变 `#EF4444 → #F97316`，显示「X折」，整数自动去小数（80 → 8折）
- 价格标签：紫蓝渐变 `#8B5CF6 → #6366F1`，显示自定义文案
- 两个字段同时存在时自动错开：折扣在上、价格标签在下
- 卡片标题行优化：
  - 有角标时标题行加 `pr-20` 右内边距，避开 badge 区域
  - 图标加 `flex-shrink-0`、文字容器加 `min-w-0` + `break-words`，长名字（如「火山方舟的Token Plan」）自动换行不再溢出
- 无角标卡片布局完全不变
